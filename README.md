## install-seurat-r-package-following-koyuki
During the seminar, I noticed that Yuka senpai and Noa senpai didn’t know how to install packages, so I wrote this script to let them install packages on their own computers. NI HA HA HA HA~
Run 'koyuki_*_install_run!.R' to install.
